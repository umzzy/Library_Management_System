# **Library Management System**
____
A console-based Library Management System built with Core Java that allows users to manage books, 
register members, issue and return books, and perform basic library operations using OOP principles 
and Java collections.

This project demonstrates the use of important Java concepts including classes, objects, encapsulation, 
inheritance, polymorphism, abstraction, arrays, lists, collections, iterators, generics, and lambdas.
It is a beginner-friendly project that helps in understanding how Object-Oriented Programming and data 
handling work together in a real-world application.
## Features
____
- Add, update, and delete books in the library
- Register new members
- Issue and return books
- Perform basic library operations
- View book details
- View member details
- Search for books and members
## Technologies Used
____
- Java SE
- Object-Oriented Programming (OOP) principles
- Java Collections Framework
